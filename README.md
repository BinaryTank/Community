# Community Repo

Community Wiki and Repo for addons


## Visit the Wiki:

https://github.com/Cloudbox/Community/wiki

## Clone this repo:

```bash
git clone https://github.com/Cloudbox/Community.git ~/community
```

## Run Ansible role:

```bash
cd ~/community
sudo ansible-playbook community.yml --tags myrole
```
